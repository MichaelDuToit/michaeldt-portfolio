Built for a local running club to upgrade their website to a more modern site. 
Easier for their Web Administrator to use when updating content. This was a full overhaul project with a custom responsive theme, HTTPS and all content from the old site carried over to the new site. 
Built utilizing GravCMS with Twig Templating, HTML, SCSS, Javascript and Gulp for the custom theme.
